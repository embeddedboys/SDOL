# sdol

#### Description
sdol(Small Device Oled Library)，是为小型mcu开发的图形库，除了包含基本的gui库函数外，还实现了一套控件库。控件库中实现了菜单、按钮、标签等基本控件，提供了简单的事件交互，此外，还实现了一套输入设备接口，用来统一输入设备事件。
